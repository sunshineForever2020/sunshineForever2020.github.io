macOS上利用 Hugo + GitHub Pages 搭建个人博客

由于平时会学习很多知识，也会遇到很多问题，但是都没有记录下来，因此想搭建个博客记录自己遇到的问题及解决办法，还有平时学到的知识的积累。

平台是macOS， 利用的工具是 Hugo 和 GitHub Pages。

以下将说明逐步搭建的过程。

Hugo

安装 Hugo

利用 homebrew 安装 hugo，

brew install hugo

建立站点

在想要搭建博客的目录下运行

hugo new site MySite

即可生成站点目录。

选择主题

登录 https://themes.gohugo.io 网点 选择你喜欢的主题，

我选择的是 Even 主题

cd ~/MySite

git clone https://github.com/olOwOlo/hugo-theme-even themes/even

可以将 themes/even/exampleSite 目录下的内容复制到MySite下看效果

回到 ~/MySite ， 利用命令

hugo server -D

可以查看本地效果

增加内容

利用命令

hugo new post/…

可以新增Markdown文件， 在里面记录内容， 也可以直接在外面写内容，复制文件到 content目录下，但是要保证头相同。

以上就是 hugo 配置内容， 还需要将其挂载到公共网域上。

GitHub Pages

创建仓库

登录 github 创建 usrname.github.io 名字的仓库。

提交内容

在 ~/MySite 目录下运行

Hugo

会自动创建 public 目录，进入该目录下， 运行

git init

git add .

git commit -m “first commit”

git branch -M main

git remote add origin git@ （以ssh的方式）

git push -u origin main

遇到问题 git push 提交错误， 提交 http 总是错误， 改为提交 ssh 解决

参考 https://sourabhbajaj.com/mac-setup/Git/README.html

在 .ssh 目录下利用

ssh-keygen -t rsa -C “your_email@example.com”

生成秘钥， 并与github账户关联将问题解决

查看 github.io

进入仓库， 在setting里面找到 github pages 进入 即可。

更新博客内容

将内容写进 content，

hugo

重新创建public

在此提交public 即可

git add .

git commit -m “”

git push -u origin main

以上即是整个搭建博客的过程！
